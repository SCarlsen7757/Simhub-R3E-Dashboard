# Simhub-R3E-Dashboard
Simhub dashboard for Raceroom Racing Experience.

This dashboard needs [Simhub R3E Extra properties plugin](https://github.com/SCarlsen7757/Simhub-R3E-Extra-properties-plugin) to work correctly. See more info there for installing that plugin.

## Import Dashboard
Download the dashboard file, and import the dashboard in Simhub, in the Dash Studio tab.
